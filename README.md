# Cyclistic Bike Share Analysis
Data analysis project using Excel, SQL and Tableau to explore Cyclistic bike share data and identify differences between casual riders and members.

### Project Overview

In this project, I analyzed Cyclistic bike share data to understand how casual riders and members use bikes differently.

### Cyclistic wants to increase the number of annual memberships.

Cyclistic wants more people to buy annual membership.
The goal is to find how casual riders behave and how to convert them into members.

### Tools & Technologies
Excel

SQL 

Tableau

### Data Preparation
Merged 12 CSV (year 2025) files into one dataset using SQL (bigquery)

Cleaned and formatted data in excel

Removed null values and inconsistencies

Created new columns (ride duration, day of week)


### Analysis Performed
Total ride trends over time

Ride duration comparison

Weekday vs weekend usage

Member vs casual rider behavior

Top 20 most visited stations

Used map to understand which bike stations are most popular

### Key Findings

Casual riders take longer rides and prefer weekends 

Casual riders take longer rides

Members take shorter, frequent rides during weekdays 

Certain stations are more popular among casual riders 

### Recommendations

Give weekend offers to casual riders

Provide free trials or discounts for membership

Show benefits of membership

Focus marketing on popular stations
